# gulp 例子


```
# 安装cnpm 使用cnpm 下载变快一点
npm install -g cnpm --registry=https://registry.npm.taobao.org

# 生成package.json文件
npm init 

# 全局安装gulp工具
cnpm install -g gulp

# 当前项目安装gulp依赖
cnpm install gulp


# 安装 gulp-stylus gulp插件
cnpm install gulp-stylus --save-dev

# 启动gulp 监听任务
gulp watch
```
